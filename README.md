## ✨ Features

- ✅ **Add/Delete tasks** with one click
- 🌓 **Dark/Light mode** (auto-saves preference)
- 🔄 **Refresh button** with confirmation
- 📱 **Fully responsive** design
- 🎨 **Sleek UI** with shadow effects
- ⚡ **Fast loading** (static Vercel deployment)
